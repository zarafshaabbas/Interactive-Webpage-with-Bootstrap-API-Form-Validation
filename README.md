# 🌐 Interactive Webpage with Bootstrap, API & Form Validation

## 📌 Project Overview

This project is a responsive and interactive webpage built using **HTML, Bootstrap, CSS, and JavaScript**. It combines modern UI styling with dynamic functionality, including API integration and form validation.

---

## ✨ Features

* Responsive layout using Bootstrap
* Clean and modern UI design
* Button to change background color dynamically
* External profile link (LinkedIn)
* GitHub API integration to display profile data
* Contact form with input validation
* Error messages for invalid inputs

---

## 🛠️ Technologies Used

* **HTML5** – webpage structure
* **Bootstrap 5** – responsive design and styling
* **CSS3** – additional custom styling
* **JavaScript** – interactivity, API handling, and validation
* **GitHub API** – fetching user profile data

## 🚀 How to Run

1. Save all files in one folder
2. Ensure:

   * `index.html`
   * `styles.css`
   * `script.js`
3. Open `index.html` in any web browser
4. Internet connection is required (for Bootstrap CDN & API)

## ⚡ Functionality

### 🔹 Background Color Change

* Click the **“Change Background Color”** button
* Page background changes randomly

### 🔹 GitHub API Integration

* Displays:

  * Name
  * Bio
  * Public repositories
  * Followers
* Data is fetched dynamically from GitHub

### 🔹 Contact Form Validation

* Validates:

  * Name
  * Email
  * Message
* Shows error messages for empty/invalid fields
